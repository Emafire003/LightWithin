## /light spoof < target > < UUID > [confirm]
**Permission required:** `lightwithin.commands.spoof`

With this command you will be able to "spoof", aka copy the entire light attributes of another player, by supplying their UUID.a
To prevent accidents, you will need to type the command again and `confirm` at the end of the command.

From version 2.0.0+
